モーダルウィンドウ

マスクを画面全体に表示させる

      
        #mask {
          position: fixed;
          top: 0;
          bottom: 0;
          left: 0;
          right: 0;
        }

モーダルを中央に配置する

      
        #modal {
          position: absolute;
          top: 40px;
          right: 0;
          left: 0;
          margin: 0 auto;
        }

クラスが付いている時モーダルを画面外上部に配置

      
        #modal.hidden {
          transform: translateY(-500px);
        }

ダブルクリックした時テキスト選択出来ないようにする

      
        #close {
          user-select: none;
        }
